# 💰 Finance

A simple finance management C# application built with .NET.

## 🚀 Features
- Expense tracking  
- Income summary  
- Data persistence  
- Clean console interface  

## 🧠 Technologies Used
- C#  
- .NET  
- Visual Studio  

## 🖥️ How to Run
1. Clone this repo:
   ```bash
   git clone https://github.com/syedzofishah/MyCSharp.git
   ```
2. Open `Finance.slnx` in Visual Studio  
3. Run the program 🎯
